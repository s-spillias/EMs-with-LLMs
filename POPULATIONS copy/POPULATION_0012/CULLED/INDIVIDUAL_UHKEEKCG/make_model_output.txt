Current Model Strengths:

 3 Has density-dependent fecundity enhancement (recently added)

Key Issues Identified:                                                                                                                          

 1 Nutrient-mediated larval survival mechanism is oversimplified: The current model uses positive SST anomalies as a proxy for nutrient runoff  
   (Equation 1b). However, this conflates two distinct processes:                                                                               
    • Warm temperatures that enhance larval development                                                                                         
    • Nutrient pulses from terrestrial runoff (typically associated with rainfall/flooding)                                                     
 2 Missing critical outbreak trigger: Literature on COTS outbreaks emphasizes that nutrient pulses from flood events are a PRIMARY trigger for  
   outbreaks, not just a modifier of larval survival. The current exponential enhancement may be too weak or poorly timed.                      
 3 Larval survival should respond to nutrient availability more directly: Rather than using temperature as a proxy, we should model nutrient    
   effects as a separate, multiplicative enhancement that can create strong episodic recruitment pulses.                                        

Proposed Improvement:                                                                                                                           

Replace the temperature-proxy nutrient mechanism with a direct nutrient-mediated larval survival boost that:                                    

 • Uses a saturating (Michaelis-Menten) response to capture threshold effects                                                                   
 • Allows for strong amplification during high nutrient conditions                                                                              
 • Separates nutrient effects from temperature effects on larval development                                                                    
 • Can create the episodic recruitment pulses needed for outbreak initiation                                                                    

This requires:                                                                                                                                  

 1 Adding a new data vector for nutrient availability (or using a proxy like rainfall/river discharge)                                          
 2 Replacing the exponential temperature-anomaly mechanism with a saturating nutrient response                                                  
 3 Adjusting parameters to allow strong but bounded nutrient enhancement                                                                        

Since we don't have a nutrient data vector available, I'll use positive SST anomalies as a refined proxy but with a saturating response function
that better captures the threshold nature of nutrient-driven recruitment pulses.

 +1. Bounds enhancement between 1x (baseline) and (1 + max_nutrient_boost)x
