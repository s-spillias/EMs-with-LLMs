 1 Are code changes needed?                                                                                                                                             

Yes. To better capture episodic outbreaks while avoiding unrealistically large recruitment pulses from extreme immigration values, I will add a biologically motivated  
saturation on larval immigration (Michaelis–Menten form). This is a single, meaningful ecological improvement aligned with the project context and avoids data leakage. 

 2 Explanation of changes                                                                                                                                               

 • Rationale: Outbreaks are often triggered by improved larval survival/settlement during favorable conditions (e.g., nutrient pulses), but recruitment is limited by   
   settlement habitat and early post-settlement bottlenecks, causing diminishing returns as larval supply increases. The current model adds immigration linearly, which 
   can over-amplify pulses and reduce realism of boom-bust timing/height.                                                                                               
 • Change: Replace linear immigration with a saturating function I_sat = I / (1 + I / I_half). This preserves units of the proxy and limits the marginal effect of very 
   large immigration. The scaling parameter I_half is the half-saturation constant (new parameter).                                                                     
 • Consequences: gamma_I retains its original units and interpretation (conversion per unit proxy). The maximum annual immigration contribution now asymptotes to       
   gamma_I * I_half, providing a realistic upper bound on how much immigrant recruitment can add in one year.                                                           
 • Implementation:                                                                                                                                                      
    • Add new parameter I_half (with bounds) to parameters.json and model.cpp.                                                                                          
    • Modify the immigration term in model.cpp to use the saturating I_sat for both t-1 and t-2 components.                                                             
    • Document the assessment and decision in intention.txt.                                                                                                            
 • Note on residuals: The residuals file was not provided here. I documented an assessment based on model structure and common COTS dynamics, and highlighted what to   
   confirm once residuals are available.
