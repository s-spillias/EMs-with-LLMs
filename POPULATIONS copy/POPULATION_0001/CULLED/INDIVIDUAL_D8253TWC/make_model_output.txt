 1 Do we need code changes?                                                                                                                                                                                                                             

Yes. To better capture boom-bust outbreaks without data leakage and to improve ecological realism, we should remove the direct adult addition from feeding (feed_gain) and instead let feeding enhance fecundity (recruitment). This aligns with COTS   
biology where nutrition primarily boosts reproduction, not immediate adult abundance.                                                                                                                                                                   

 2 Summary of needed changes                                                                                                                                                                                                                            

 • Model fit assessment and context:                                                                                                                                                                                                                    
    • Current structure includes Allee effect, SST modifier, coral-dependent carrying capacity, and feeding losses on coral. This can generate busts via resource depletion and recruitment limitation via K_A, but the adult “feed_gain” term likely   
      smooths and advances booms unrealistically by adding adults directly from feeding, potentially mis-timing peak outbreaks.                                                                                                                         
    • Missing/oversimplified: nutrition-to-reproduction linkage (currently misrepresented as adult addition), no predator/disease feedback, linear K_A vs coral, no explicit runoff/nutrient driver beyond exogenous cotsimm_dat.                       
 • Ecological improvement implemented:                                                                                                                                                                                                                  
    • Shift feeding from adding adults to boosting fecundity via a saturating per-capita feeding effect. This preserves a biologically plausible pathway: better feeding -> higher fecundity -> stronger recruitment next step, aiding realistic timing 
      and magnitude of outbreaks while allowing sharp busts when coral declines.                                                                                                                                                                        
    • Added parameters:                                                                                                                                                                                                                                 
       • beta_feed_A: max fractional fecundity boost from feeding (dimensionless).                                                                                                                                                                      
       • feed_half_A: half-saturation of per-capita feeding rate for fecundity (yr^-1).                                                                                                                                                                 
 • No current-time data leakage is introduced; all processes use previous-step predictions or exogenous inputs.                                                                                                                                         
 • Soft bounds added for new parameters; previous eta_A is now effectively deprecated (kept in parameters.json for compatibility but no longer used).
