 2 Alignment to the project context                                                                                                                                                                                             

 • Strengths: The model already captures core boom-bust mechanisms: prey-dependent COTS survival, temperature-modulated reproduction, Allee effect on recruitment, prey-dependent recruitment saturation (via kc_carry),        
   multi-prey functional response with preference, and bleaching impacts on coral. Immigration forcing allows exogenous outbreak initiation.

   that accelerate busts. The current survival only depends on prey and a baseline s0_cots, which can make declines overly gradual if prey remains somewhat available.

 • Temperature-dependent adult survival: Missing; important for realistic bust termination during hot anomalies/bleaching periods.

 • Add a temperature-driven adult survival penalty linked to the same logistic heat stress (bleach_level) already used for coral bleaching:
