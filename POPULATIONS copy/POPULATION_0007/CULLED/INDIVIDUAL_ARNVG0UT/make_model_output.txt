 • Chosen ecological improvement:                                                                                                               
    • Add a latent phytoplankton state P_t as a bottom-up resource driver for recruitment. P_t follows a simple logistic growth modulated by    
      temperature (a Gaussian thermal performance curve), with linear loss, and feeds into recruitment via a Monod (saturating) limitation term:
       • P dynamics (deterministic, using t-1 SST to avoid leakage): P_t = P_{t-1} + rP * P_{t-1} * (1 - P_{t-1}/K_P) * exp(-beta_P * (SST_{t-1}
         - T_opt_P)^2) - muP * P_{t-1}                                                                                                          
       • Recruitment modifier: f_food = (P_{t-1} / (K_food + P_{t-1}))^p_food                                                                   
    • This keeps the model relatively simple while adding the crucial resource limitation link needed by the project context, and avoids using  
      current time step observed variables (no data leakage).
