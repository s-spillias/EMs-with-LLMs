 1 Are code changes needed?                                                                                                                                                                                                     

Yes. I propose adding resource-limited adult mortality to strengthen bust dynamics after coral depletion, which is central to episodic outbreak behavior.                                                                       

 2 Explanation of changes                                                                                                                                                                                                       

 • Rationale: In the current model, adult mortality is baseline plus density-dependent. However, when coral cover drops after heavy predation, adults should experience elevated mortality due to starvation/food limitation.   
   This feedback sharpens and shortens the post-peak decline (bust), improving realism for boom-bust cycles.                                                                                                                    
 • Change: Add a new parameter mu_food (yr^-1) that increases adult mortality when food is scarce, using the existing saturating food index f_food = R/(K_R + R). Adult mortality becomes:                                      
   Mort_adult = [muC + gammaCC_prev + mu_food(1 - f_food)] * C_prev                                                                                                                                                             
   This is zero when food is plentiful (f_food ≈ 1), and maximal when food is scarce (f_food ≈ 0).

    • model.cpp: add PARAMETER(mu_food), add range penalty, and use it in Mort_adult.
