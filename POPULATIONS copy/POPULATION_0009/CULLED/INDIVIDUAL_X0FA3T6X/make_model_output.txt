Yes. The current model removes phytoplankton biomass through grazing, but only the assimilated fraction AE_Z*grazing enters zooplankton biomass. The unassimilated fraction (1 - AE_Z)*grazing is currently lost from the system
unless captured indirectly by ex_Z or mortality. This breaks mass balance and typically leads to biased post-bloom nutrient dynamics (e.g., underpredicted N rebound and distorted P/Z timing). To address this and improve     
ecological realism within the given mixed-layer NPZ context, I propose adding an explicit “sloppy feeding/egestion to dissolved nutrients” pathway: route a fraction r_egN of the unassimilated grazing directly to the         
dissolved nutrient pool.                                                                                                                                                                                                        

 2 Explanation of changes                                                                                                                                                                                                       

 • Ecological rationale: Sloppy feeding and egestion rapidly recycle a portion of ingested but unassimilated phytoplankton into dissolved nutrients. In mixed layers, this pathway helps sustain secondary peaks and modulates  
   bloom decay timing. Without it, the model can systematically underpredict N after grazing events and misfit the P/Z phase relationship.                                                                                      
 • Mathematical change: Add egest_to_N = r_egN * (1 - AE_Z) * grazing to the remineralization term feeding N, so                                                                                                                
   remin_N = r_PNmortP + r_ZNmortZ + excrZ + egest_to_N.                                                                                                                                                                        
 • Parameters: Introduce r_egN in [0,1] with a weak smooth penalty to keep it within plausible range. Default initial value 0.9 (most unassimilated ingestion goes quickly to dissolved nutrients), but the model will estimate 
   it.                                                                                                                                                                                                                          
 • Diagnostics: Expose egest_to_N as a reported diagnostic.                                                                                                                                                                     
 • Non-leakage: State updates continue to use only previous-step states.
