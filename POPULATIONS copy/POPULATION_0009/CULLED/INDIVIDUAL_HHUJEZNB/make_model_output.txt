 1 Do we need code changes?                                                                                                                                                                                                     

 • Yes. To better reflect mixed-layer light dynamics in the project context, I added phytoplankton self-shading so light limitation is dynamic rather than a fixed constant.                                                    

 2 What changed and why?                                                                                                                                                                                                        

 • Assessment summary:                                                                                                                                                                                                          
    • Model fit: I can’t quantitatively evaluate fit because the residuals file was not provided. Please share it so I can compute and plot residual patterns and quantify misfit.                                              
    • Project context: The model captures core NPZ interactions, temperature effects (Q10), dilution/exchange, ingestion partitioning, and lognormal observation noise. It’s appropriate for mixed-layer NPZ dynamics.          
    • Oversimplifications: Light limitation is currently fixed (L_avail), which ignores self-shading and prevents feedbacks that help stabilize bloom dynamics. Quadratic Z mortality (gamma_Z) is not temperature-modified;    
      this may be acceptable but is a potential future refinement.                                                                                                                                                              
    • Updated parameters: Q10 has a literature-updated value and range. Growth, linear mortalities, excretion, and grazing already use this temperature modifier. Quadratic Z mortality remains unscaled; I’m not changing it   
      now to keep to one focused improvement.                                                                                                                                                                                   
 • Chosen improvement:                                                                                                                                                                                                          
    • Add phytoplankton self-shading using a Beer–Lambert style attenuation: I_eff = L_avail * exp(-k_I * P_prev). Then fL = I_eff / (K_L + I_eff).                                                                             
    • Ecological rationale: As phytoplankton biomass increases, light in the mixed layer decreases due to self-shading, limiting further growth; this commonly stabilizes bloom peaks and improves phase relationships among    
      N-P-Z.                                                                                                                                                                                                                    
    • Complexity: Adds one parameter (k_I) with clear interpretation and conservative bounds.                                                                                                                                   
 • Implementation details:                                                                                                                                                                                                      
    • New parameter k_I (units m^3 g C^-1) controls the strength of self-shading.                                                                                                                                               
    • Light limitation fL is computed from previous-step P only, preserving the no data-leakage rule.                                                                                                                           
    • Added diagnostic output L_eff_pred to track effective light through time.                                                                                                                                                 
    • Added a gentle bounds penalty for k_I.
