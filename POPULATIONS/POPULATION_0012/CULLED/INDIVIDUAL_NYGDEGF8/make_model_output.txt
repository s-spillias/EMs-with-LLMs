Assessment & Proposed Improvement:                                                                                                                                                                                     
Residual analysis suggests that the current logistic outbreak trigger is too smooth and does not fully capture the threshold (boom-bust) dynamics of COTS outbreaks. To better reflect the rapid switch‐like transition
observed ecologically, we replace the logistic function by a Hill (power) function. This change introduces a new parameter (outbreak_exponent) that controls the non‐linearity: higher values yield a more abrupt      
outbreak trigger. We also use an environmental modifier multiplicatively on the COTS population when computing the Hill ratio. Note that we must avoid using current time step observations (i.e. *_dat values) in     
predictions, so we are keeping to the previous time step predictions.
