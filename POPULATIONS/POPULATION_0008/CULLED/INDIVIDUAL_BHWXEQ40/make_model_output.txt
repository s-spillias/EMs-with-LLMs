Below is our proposed update. We decided to improve the ecological realism by making the conversion efficiency for COTS growth vary with coral availability. In the revised model the efficiency term becomes dynamic: it is now scaled by the total coral cover (computed as the sum of predicted fast‑ and slow‑coral) via a Michaelis–Menten (saturating) function. This            
modification is ecologically justified because predatory efficiency should increase with prey (coral) abundance and taper off when prey becomes overly abundant. To allow the model to flexibly describe this effect, we add a new parameter, half_saturation_coral, that sets the coral cover at which efficiency reaches half its maximum value. Note that all prediction            
calculations in the time loop now strictly use previous time step values to avoid data leakage.

We add a new parameter, "half_saturation_coral", that defines the coral cover at which the effective efficiency is half of its maximum. This added complexity is ecologically justified as it better represents resource‐limited predation and prey switching behaviors  
 coral reef systems.
